# Blog SEO Tools

An open-source project focused on helping bloggers and content creators improve their SEO, writing workflow, and productivity.

## 🚀 Features
- SEO-friendly title ideas
- Blog content structure guidance
- Keyword planning ideas
- Content optimization tips

## 🎯 هدف (Goal)
To build simple and practical tools for bloggers, especially beginners, so they can grow faster without expensive tools.

## 📌 Roadmap

- Create content templates
- Build simple keyword suggestion tool
